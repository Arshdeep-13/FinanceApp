## Budgie app

Budgie app is a tool to track your expenses and analyze them, built with React and styled with Tailwind css.

## Demo (Desktop only)
https://budgie-app-yusuflisawi.netlify.app/

---

## Installation

```plaintext
git clone git@github.com:YusufLisawi/budgie.git
cd budgie
npm install
npm start
```

## Tools used

-   Reactjs
    -   Context api
    -   useState, useEffect
    -   react router v6
-   LocalStorage
-   Tailwind css
-   chart.js
-   SweetAlert2
-   react-tiptool
-   react-flipMove
